# TUIShogi - Text User Interface Shogi

## Requirements

  * OpenShogiLib

## Installation

    % sudo apt-get install libosl-dev
    % git clone https://github.com/myokoym/tuishogi
    % cd tuishogi
    % ./autogen.sh
    % ./configure
    % make
    % cd src
    % make tuishogi

## Launch

    % ./tuishogi
